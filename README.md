# How to use Docker and Docker Compose for PHP development

## How to use the presentation

- Install the dependencies:
  ```bash
  $ yarn install --frozen-lockfile --check-files
  ```

- Build the artifacts:
  ```bash
  $ yarn build
  ```

- Serve the presentation:
  ```bash
  $ yarn preview
  ```
