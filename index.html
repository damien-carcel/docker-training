How to run a Docker container



Run a container in interactive mode

  • Pull and run the Debian slim image

  $ docker pull debian:jessie-slim
        

  $ docker images
        

  $ docker run -i -t --rm debian:jessie-slim bash

Explain "pull" and the tag system

"docker images" same thing that "docker image ls"

Before image name: options; after image name: command that will run in the container

How to run a Docker container



Run a container in detached mode


  $ docker run -d nginx
        

  $ docker ps
        

  $ docker ps -a

"docker ps" same thing that "docker container ls"

How to run a Docker container



A more usefull example


  $ touch /tmp/index.html
        

  $ docker run -d --name nginx-container -p 8001:80 -v /tmp/index.html:/usr/share/nginx/html/index.html:ro nginx
        

  $ docker ps
  $ docker exec nginx-container cat /usr/share/nginx/html/index.html
  • Access the server through localhost:8001

Summary

Command Usage
docker pull Pull an image from a repository
docker images List all images on your machine
docker ps [-a] List all containers on you machine
docker run -i -t --rm image command Run a container in interactive mode
docker run -d image Run a container in detached mode
docker exec container_name command Execute a command inside a running container
docker rm [-f -v] container_name|id Delete a container
docker rmi [-f] image_name|id Delete an image
docker system prune --volume Clean everything

Docker compose

Tool for defining and running multi-container Docker applications

Based on a YAML file

Docker compose



# docker-compose.yml
version: '2'

services:
  akeneo:
    image: akeneo/php:5.6
    volumes:
        - ./:/home/docker/application
    networks:
      - akeneo

  mysql:
    image: mysql:5.5
    ports:
        - 33006:3306
    networks:
      - akeneo

networks:
  akeneo: ~
